Array Methods

1.concat()

this method used to return the new array by joining two or more arrays.

2.slice()

this method used to extract the section of array and returns the new array.

3.join()

this method used to join all the elements in an array into string.

4.pop()

this method used to remove last elements from an array and returns that element.

5.push()

this method used to add elements at the last and returns the new array.

6.unshift()

this method used to add elements at the beginning and returns the new array.

7.toString()

this method used to convert the array into string.

8.sort()

this method used to sort and returns the sorted array.

9.splice()

this method used to change the contents of array adding new elements while removing old elements.

10.reverse()

this method used to reverse the contents of array and return the new array.

11.shift()

this method used to remove elements at the beginning and returns the new array.

12.lastIndexOf()

this method used to return index of the last occurance of the element and returns -1 if not found.

13.indexOf()

this method used to return index of the first occurance of the element and returns -1 if not found

String Methods

1.charAt()

this method used to return the character at the specified index.

2.concat()

this method used to add two or more string and return the new string.

3.valueOf()

this method used to return the primitive value of string object.

4.toLowerCase()

this method used to convert the string into lowercase .

5.toUpperCase()

this method used to convert the string into uppercase .

6.substring()

this method used to returns the new substring on given parameters.

7.split()

this method used to split string object into array.
